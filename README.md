# pyFinance

This project provides a Python module named Group_21 for financial data analysis and visualization. It allows you to:

Fetch historical data from Yahoo Finance and NSE (India).
Calculate valuation ratios (P/E, P/B, P/S, PEG).
Visualize data with line plots, candlestick charts, and technical indicators (Moving Averages, MACD, RSI, Volatility).
Export data and reports in various formats (PDF, CSV, Excel, etc.).
